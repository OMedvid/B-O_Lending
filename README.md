# B&O Landing

A responsive landing page for a **Bang & Olufsen** product website built with **HTML** and **SCSS**.

The project presents a modern and visually clean interface focused on premium audio products. It includes a full-screen hero section, navigation menu, product highlights, recommended categories, contact information, and other structured landing page sections designed for smooth user interaction and readability.

---

## Features

- Responsive landing page layout
- Full-screen hero section
- Burger menu navigation
- Product presentation sections
- Recommended categories block
- Contact information section
- Smooth page structure and visual hierarchy
- Clean and modern UI

---

## Technologies Used

- HTML5
- SCSS
- BEM methodology
- JavaScript
- Gulp
- ESLint
- Stylelint
